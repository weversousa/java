# VARIÁVEL E CONSTANTE

* Case-sensitive.
* `camelCase` para variável.
* `SCREAMING_SNAKE_CASE` para constante.

```java
package fundamentos;

public class VariavelConstante {

    public static void main(String[] args) {

        // Variável
        double raio = 3.4;

        // Constante
        final double PI = 3.14159;
    }
}
```
